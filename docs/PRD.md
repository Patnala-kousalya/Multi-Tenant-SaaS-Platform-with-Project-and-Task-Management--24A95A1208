## User Personas

### Super Admin
- Manages all tenants

### Tenant Admin
- Manages users & projects

### User
- Works on tasks

## Functional Requirements
FR-01: System shall allow tenant registration  
FR-02: System shall allow login  
FR-03: System shall allow user creation  
FR-04: System shall allow project creation  
FR-05: System shall allow task management  

## Non-Functional Requirements
NFR-01: Secure authentication  
NFR-02: Scalable architecture  
NFR-03: Fast response time  
NFR-04: Data isolation  
NFR-05: Docker support
